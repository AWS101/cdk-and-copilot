# Copilot

## Requisitos
* Entorno Cloud9 para ejecutar el workshop
* [Copilot CLI](https://aws.github.io/copilot-cli/docs/getting-started/install/)


## Desarrollo


```
copilot env init
copilot env deploy --name test
copilot app init
copilot svc init
copilot storage init
  Table Name: hits
```
